# Artem — Magnit Ambassador (Landing)

Одностраничный сайт-визитка для GitHub Pages.

## Быстрый деплой на GitHub Pages
1. Создай репозиторий на GitHub (например `artem-landing`).
2. Загрузи **все файлы** из папки проекта в корень репозитория.
3. Settings → Pages → Deploy from a branch → `main` / `/ (root)`.
4. Готово — GitHub выдаст ссылку.

## Что заменить под себя
В `index.html` в разделе **Контакты** замени ссылки:
- Telegram
- Email
- VK
- GitHub

## Структура
- `index.html` — контент
- `styles.css` — дизайн и адаптив
- `script.js` — интро, reveal-анимации, parallax
- `assets/` — изображения
